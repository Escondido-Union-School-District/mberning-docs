What This Tool Does

Finds the Student Resources and Parent Resources pages on every EUSD school's Thrillshare CMS site — even though those pages are named inconsistently across schools (e.g. "Students", "Resources for Students", "Parents and Families") — and adds an accessible text caption beneath every graphical button on those pages. The caption is a centered text link that points to the same destination as the graphic, so screen-reader and keyboard users get a real, selectable label.

Why this matters: Image-only navigation buttons fail WCAG 2.1 and were flagged in the EUSD DOJ Web Rule audit. Adding a captioned text link preserves the visual design while giving assistive tech something meaningful to announce.

Prerequisites

One-Time Setup

  1. Python 3.10+ installed
  2. Playwright with Chromium installed (playwright install chromium)
  3. Project folder: ~/projects/eusd/thrillshare-page-edit/
  4. Thrillshare access — an EUSD Google account with CMS edit permissions on the schools you intend to update

If Setting Up on a New Machine

# 1. Clone the repo
git clone https://github.com/Escondido-Union-School-District/thrillshare-page-edit.git
cd thrillshare-page-edit

# 2. Install dependencies
pip install -r requirements.txt
playwright install chromium
