ADA Presentations

User Guide — Escondido Union School District

Last updated: 2026-04-09 — Initial guide created.

What This Tool Does

ADA Presentations is a local Flask web app that converts inaccessible PDF presentations (typically exported from Google Slides) into ADA / WCAG 2.1 AA compliant HTML slide decks branded for EUSD. It uses the Claude Vision API to analyze slide images, generate alt text, and reconstruct tables, then assembles the result into a reveal.js-based slide deck.

Requirement: An ANTHROPIC_API_KEY must be set in .env at the project root for the AI enrichment step to work.

Prerequisites

One-Time Setup (already done on your machine)

  1. Python 3.14 installed
  2. Project folder: ~/projects/eusd/ada-presentations/
  3. Dependencies installed via pip install -r requirements.txt
  4. API key set in .env: ANTHROPIC_API_KEY=sk-ant-...

If Setting Up on a New Machine

# 1. Clone the repo
git clone https://github.com/Escondido-Union-School-District/ada-presentations.git
cd ada-presentations

# 2. Install dependencies
pip install -r requirements.txt

# 3. Create a .env file with your Anthropic API key
echo "ANTHROPIC_API_KEY=sk-ant-..." > .env

Step-by-Step

1 Start the web app

python run.py

The app starts at http://localhost:5000.

2 Upload a PDF

Open the app in your browser and upload the source PDF presentation. A new project folder is created under ~/projects/eusd/ada-presentations/projects/.

3 Review and enrich

The app extracts each page, sends slide images to the Claude Vision API for alt text and table reconstruction, and saves the enriched content as JSON.

4 Generate the accessible HTML deck

When enrichment is complete, the app assembles a reveal.js-based slide deck with EUSD branding. Download the result from the project page.

Understanding the Output

Each uploaded PDF becomes a project folder under projects/ containing the extracted page images, enrichment JSON, and the final HTML slide deck. The deck uses reveal.js with EUSD branding (royal blue #1B3A8C, accent blue #1a56db, highlight gold #f9a825, Arial font) and meets WCAG 2.1 AA: semantic headings, alt text on every image, keyboard navigation, and accessible tables.

Troubleshooting

ErrorCauseFix
ANTHROPIC_API_KEY not set Missing or empty .env file Create .env at project root with ANTHROPIC_API_KEY=sk-ant-...
Address already in use Port 5000 is occupied by another process Stop the other process, or change the port in run.py
ModuleNotFoundError Dependencies not installed Run pip install -r requirements.txt
